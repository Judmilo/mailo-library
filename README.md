# 📚 Mailo Library

An **Online Library web application** that allows users to search for books and explore books by categories.  
The application fetches book data from the Open Library API and displays information such as titles, authors, and book covers.

---

## 🚀 Features

- 🔎 Search books by title
- 📚 Browse books by categories
- 👨‍💻 View book covers and basic information
- ⚡ Fast and responsive interface
- 📱 Mobile-friendly design

---

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- CSS / TailwindCSS
- Open Library API

---

## 📡 API

This project uses the Open Library API to fetch book data.

API Documentation:  
https://openlibrary.org/developers/api
